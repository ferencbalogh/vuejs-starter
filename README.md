# VueJs Starter 1.0.0

This is a vuejs starter project.

## What's included:

- babel
- typescript
- e2e-nightwatch
- scss (node)
- bootstrap (node)
- ion icons

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
yarn run watch
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```
